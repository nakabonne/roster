# roster

A cli tool to add values to google spreadsheets

# Usage

```
$ roster -c A1 -v hello! -i 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms
```

|option|description|
|:------|:----------|
|-c|Set cell|
|-v|Set value|
|-i|Set Your Spreadsheet id|

# Installation

```
go get -u github.com/ryonakao/roster
```
